Privacy Policy

We respect your privacy and are committed to protecting your personal data.

1. Information We Collect

We may collect the following information when you use our app:

  • Name, email address, and login credentials
  • Location data (including background location when enabled)
  • Camera input for face detection used for attendance marking

2. How We Use Information

We use collected information for the following purposes:

  • To create and manage user accounts
  • To provide attendance marking functionality
  • To verify user identity using face detection
  • To track attendance using location services

3. Location Data

Our app may access location data, including background location, to verify attendance and user presence. Location data is used only for core app functionality and is not shared with third parties for marketing purposes.

4. Face Detection

The app uses camera-based face detection to mark attendance. Face processing is performed in real-time for verification purposes. We do not use facial data for advertising or unrelated purposes.

5. Data Storage

We store only necessary user data required for app functionality. Sensitive data is protected using appropriate security measures.

6. Data Sharing

We do not sell or rent user data. We may share limited data only when required by law or to provide core services.

7. Security

We take reasonable steps to protect your data from unauthorized access or misuse.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page.
